Mis trabajos

Desafio 1 - Mi curriculum Vitae en HTML

Aqui se evaluo el conocimiento de la estructura del lenguaje de programación y el correcto uso la estructura básica de un archivo HTML, incluyendo doctype, html, head y body. Logrando que el contenido se viera organizado y fácil de leer para los usuarios.

Desafio 2 - Construyendo una Lading age

En esta evaluación aprendimos cómo integrar hojas de estilo (CSS) en un proyecto web, y luego crear una estructura de un proyecto guardando nuestros archivos CSS en una carpeta diseñada para eso, con el fin de tener más herramientas para implementar diseños web efectivos.

Desafio 3 - Iguana Page

Aplicamos algunos conceptos básicos como el de propiedad display y el valor flexbox. Finalmente, profundizamos en HTML semántico, el cual permitio hacer código más entendible para los navegadores, así como también para otros desarrolladores del mismo sitio.

Desafio 4 - Cupón

En esta desafio aprendimos qué son los frameworks CSS, cuáles son las ventajas de su utilización y de Bootstrap y uno de sus usos más comunes, la distribución de elementos basados en su grilla.

Desafio 5 - Github

Se realizaron operaciones de navegación de directorios, usando los comandos básicos del terminal, para crear y manipular archivos y directorios. Inicializar un proyecto con Git y realizar un fork de un proyecto en github.

Prueba - Desarrollo de portafolio

Aplicar todos los conocimientos adquiridos durante el desarrollo del curso dejándolo disponible en Github Pages. Finalmente, trabajar de forma colaborativa en Github haciendo alguna contribución a un repositorio de algún compañero.
